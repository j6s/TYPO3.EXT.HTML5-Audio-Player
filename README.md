HTML5 Audio Player
==================

HTML5 audio player with knob.

How to embed
------------
You can embed audio using the following code.

```html
<source type="audio/mp3" src="YourSong.mp3">
```

Don't forget the .ogg file, it`s needed for Firefox 3.6+ and Opera 10.5+
